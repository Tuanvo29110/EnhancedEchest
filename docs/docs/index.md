# Welcome to EnhancedEchest

**EnhancedEchest** is a free, open-source Minecraft plugin that upgrades the vanilla ender chest into a larger, persistent, multi-chest storage system. Players get ender chests of up to **54 slots**, can own several at once, and manage them from a clean in-game menu.

## What EnhancedEchest Adds

<CardGrid>

<FeatureCard icon="Package" title="More Space">
The vanilla ender chest holds 27 items. EnhancedEchest gives each player a configurable chest of up to 54 slots, opened from the same ender chest block or the <code>/ec</code> command.
</FeatureCard>

<FeatureCard icon="Archive" title="Multiple Chests Per Player">
Players are no longer limited to a single ender chest. Admins can grant extra chests, each with its own size and custom name, switchable from an in-game management menu.
</FeatureCard>

<FeatureCard icon="Palette" title="Custom Names and Icons">
Each chest can have its own name and icon chosen from an in-game picker - no commands needed. Named chests show their name as the inventory title.
</FeatureCard>

<FeatureCard icon="Key" title="Permission-Based Grants">
Hand out extra chests by rank using permission nodes. Grants sync on open and are revoked cleanly when a rank changes, spilling items to a recoverable temporary chest.
</FeatureCard>

</CardGrid>
